# PWA

## Get Started
1. Install Node.js
2. Install all dependencies (`npm install`)
3. Install Firebase CLI (`npm install -g firebase-tools`)
4. Login Firebase (`firebase login`)
5. Start the server (`npm start`)

### Deployment
1. Build (`npm run build`)
2. Deploy (`firebase deploy`)
